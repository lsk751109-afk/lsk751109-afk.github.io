주식회사 제일솔루션 홈페이지 — 가비아 업로드 안내
====================================================

1. 회사 연락처 입력 (필수)
--------------------------
company-config.js 파일을 메모장으로 열어 아래 따옴표 안에 실제 정보를 입력합니다.

  phone: "대표전화",
  mobile: "문의 담당 휴대전화",
  contactTitle: "문의 담당자 직책",
  email: "문의 받을 이메일",
  address: "사업장 주소",
  businessHours: "상담시간",

입력하지 않은 전화·이메일·주소 항목은 홈페이지에서 자동으로 숨겨집니다.
이메일을 입력하면 문의 버튼이 자동으로 '이메일 문의 보내기'로 바뀝니다.


2. 가비아 웹호스팅 업로드
-------------------------
가비아 My가비아 → 서비스 관리 → 웹호스팅 → FTP 접속정보를 확인합니다.
FileZilla 또는 가비아 웹FTP로 이 폴더 안의 모든 파일과 assets 폴더를
웹 루트 폴더(일반적으로 www 또는 public_html)에 업로드합니다.

중요: jeil-solution-gabia 폴더 자체가 아니라, 폴더 '안의 내용'을 올려야
도메인 첫 화면에서 index.html이 바로 열립니다.


3. 업로드 후 확인
-----------------
□ PC에서 메인 화면 확인
□ 휴대폰에서 PC용 전체 화면이 축소되어 표시되는지 확인
□ 메뉴 이동과 설비 사진 확대 기능 확인
□ 대표전화·이메일·주소 확인
□ 문의폼 입력 후 이메일 프로그램 실행 확인
□ HTTPS(보안서버) 적용 여부 확인


4. 문의폼 안내
---------------
이 홈페이지의 문의폼은 별도의 서버 데이터베이스를 사용하지 않습니다.
company-config.js에 이메일을 입력하면 방문자의 이메일 프로그램을 열어
문의 내용을 보내는 방식입니다. 홈페이지에서 직접 자동 발송하려면
가비아 호스팅 환경에 맞는 PHP 메일 또는 외부 폼 서비스를 추가해야 합니다.


5. 파일 구성
-------------
index.html        홈페이지 본문
styles.css        디자인·반응형 화면
script.js         메뉴·사진 확대·문의 기능
company-config.js 회사 연락처 설정
assets/           설비 사진·글꼴·파비콘
robots.txt        검색엔진 접근 설정

제작 기준: PC용 고정형 웹 / 최신 Chrome, Edge, Safari / UTF-8
스마트폰 전용 반응형 레이아웃은 적용하지 않았습니다.
